# Instrument and configure pods [apm-get-started-webhook]

To instrument and configure your application pods, complete the following steps:

1. [Add the helm repository to Helm](/reference/apm-webhook-add-helm-repo.md)
2. [Configure the webhook with a Helm values file](/reference/apm-webhook-configure-helm.md)
3. [Install the webhook with Helm](/reference/apm-webhook-install-helm.md)
4. [Add a pod template annotation to each pod you want to auto-instrument](/reference/apm-webhook-add-pod-annotation.md)
5. [Watch data flow into the {{stack}}](/reference/apm-webhook-watch-data.md)
